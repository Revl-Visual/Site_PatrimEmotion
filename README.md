# PatrimEmotion® — Site Web

Site vitrine Next.js 15 (TypeScript, Tailwind). Contenu statique, sans CMS ni variables d’environnement obligatoires.

## Prérequis

Node.js 18.18+ ou 20 LTS (voir `.nvmrc`).

## Commandes

```bash
npm ci
npm run dev      # http://localhost:3000
npm run build
npm start
```

## Déploiement Infomaniak

Voir [DEPLOIEMENT_INFOMANIAK.md](./DEPLOIEMENT_INFOMANIAK.md).

## Structure

- `app/` — pages
- `components/` — composants
- `lib/` — URLs et branding
- `public/` — images, PDF, vidéos, polices
